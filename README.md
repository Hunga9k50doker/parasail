## ✎ᝰ. RUNNING

- Clone Repository

```bash
git clone https://github.com/Hunga9k50doker/parasail.git
cd parasail
```

- Install Dependency

```bash
npm install
```

- Setup config in .env

```bash
nano .env
```

- Setup input value

* proxy: http://user:pass@ip:port

```bash
nano proxy.txt
```

- privateKey: how to get privateKey => join my channel: https://t.me/airdrophuntersieutoc

